# 🤖 Coze扣子平台APP下载二维码查询指南

## 📋 在Coze平台上查找APP下载二维码

### 方法一：在Coze工作台查找

#### 第1步：登录Coze工作台

1. 访问：https://www.coze.cn/
2. 使用你的账户登录

#### 第2步：进入你的工作空间

1. 点击左上角的个人头像
2. 进入你的工作空间

#### 第3步：找到已部署的Bot/应用

1. 在左侧导航栏，点击 **"Bot"** 或 **"我的Bot"**
2. 找到你之前部署的Bot/应用
3. 点击进入详情页

#### 第4步：查看发布和分发

1. 在Bot详情页，找到 **"发布"** 或 **"分发"** 标签
2. 查看已发布的平台：
   - 钉钉
   - 飞书
   - 微信
   - Slack
   - 网页版

#### 第5步：获取二维码

根据发布的平台不同，获取方式也不同：

**如果是网页版**：
- 点击 **"网页版"** 或 **"Web"**
- 会显示访问链接
- 可以生成二维码分享

**如果是移动端APP**：
- 点击 **"移动端"** 或 **"App"**
- 会显示二维码
- 扫描二维码即可在手机上使用

---

### 方法二：查看Bot的分享功能

#### 第1步：打开Bot详情页

1. 在Coze工作台找到你的Bot
2. 点击进入

#### 第2步：找到分享按钮

1. 查看右上角或顶部的 **"分享"** 按钮
2. 点击分享

#### 第3步：选择分享方式

1. **分享链接**：复制链接，可以生成二维码
2. **分享到**：选择平台（微信、钉钉等）
3. **生成二维码**：点击生成二维码

---

### 方法三：在已发布的平台查找

如果你的Bot已经发布到第三方平台：

#### 钉钉
1. 打开钉钉APP
2. 搜索你的Bot名称
3. 或者在工作台找到

#### 飞书
1. 打开飞书APP
2. 搜索你的Bot名称
3. 或者在应用中心找到

#### 微信
1. 打开微信
2. 扫描Bot分享的二维码
3. 添加到微信小程序或公众号

## 📱 Coze Bot的不同形态

### 1. 网页版Bot

**访问方式**：
- 直接通过浏览器访问
- 分享链接

**生成二维码**：
1. 复制网页链接
2. 使用二维码生成工具

2. 点击发布或查看
3. 显示二维码

### 3. 微信小程序

**访问方式**：
- 扫描二维码
- 微信搜索

**获取二维码**：
1. 在Bot详情页找到 **"微信"**
2. 点击查看
3. 显示小程序二维码

## 📊 Coze平台特性

### 发布渠道

Coze支持发布到多个渠道：

| 渠道 | 说明 | 二维码位置 |
|------|------|-----------|
| **网页版** | 浏览器访问 | Bot详情 → 分享 |
| **钉钉** | 钉钉工作台 | Bot详情 → 发布 → 钉钉 |
| **飞书** | 飞书工作台 | Bot详情 → 发布 → 飞书 |
| **微信** | 微信小程序 | Bot详情 → 发布 → 微信 |
| **Slack** | Slack集成 | Bot详情 → 发布 → Slack |

### Bot状态

查看Bot的发布状态：

| 状态 | 说明 | 是否有二维码 |
|------|------|-------------|
| **已发布** | 已上线 | ✅ 有 |
| **草稿** | 未发布 | ❌ 无 |
| **审核中** | 等待审核 | ⏳ 有预览二维码 |
| **已下线** | 已下架 | ❌ 无 |

## 🔧 如果找不到二维码

### 检查1：确认Bot已发布

1. 进入Bot详情页
2. 查看状态标签
3. 如果是"草稿"状态，需要先发布

### 检查2：查看发布记录

1. Bot详情页 → "发布"标签
2. 查看发布历史
3. 点击已发布的记录

### 检查3：联系Coze支持

如果以上方法都找不到：
1. Coze工作台 → 右下角帮助
2. 提交工单
3. 说明：找不到Bot下载二维码
