## 🚀 立即部署（3步完成）

### 方法：拖拽部署到Netlify（最简单）⭐

**第1步：打开Netlify部署页面**
```
https://app.netlify.com/drop
```

**第2步：上传文件**
- 找到文件夹：`/workspace/projects/client/dist`
- 将整个 `dist` 文件夹拖拽到页面上的上传区域
- 等待上传完成（约30秒）

**第3步：获取访问链接**
- 上传完成后会显示访问链接
- 复制链接并分享给用户

## ⚙️ 重要配置

部署后必须配置后端API地址：

1. 登录Netlify，进入项目设置
2. 找到"Environment variables"
3. 添加环境变量：
   ```
   Key: EXPO_PUBLIC_BACKEND_BASE_URL
   Value: http://your-backend-domain.com:9091
   ```
