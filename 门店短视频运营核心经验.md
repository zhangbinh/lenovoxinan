# 门店短视频运营核心经验
===========================

## 一、爆款文案公式

### 小红书爆款结构
【封面】：人+产品+情绪
【标题】：数字+悬念+利益
【正文】：场景代入→痛点共鸣→产品价值→行动引导
【互动】：设问征集，引导评论

### 抖音脚本结构（黄金5秒）
前3秒：强钩子（冲突/悬念/数字）
3-15秒：核心卖点展示
15-60秒：使用场景+痛点解决
结尾：互动引导+关注引导

## 二、内容加热判断标准

### 小红书（薯条）
| 互动率 | 建议 | 预算 |
|--------|------|------|
| >2.5% | 薯条加热 | 50-150元 |
| 1%-2.5% | 低预算测试 | 30-80元 |
| <1% | 暂不加热，优化内容 | - |

### 抖音
| 条件 | 建议 | 工具 | 预算 |
|------|------|------|------|
| 互动率>3% 且点赞>100 | 规模化放大 | 巨量本地推 | 100-300元/天 |
| 互动率1.5%-3% | DOU+测试 | DOU+ | 50-100元 |
| 互动率0.8%-1.5% | 低预算探路 | DOU+ | 30-50元 |
| <0.8% | 优化内容，暂不投流 | - | - |

## 三、到店转化提升

### 转化链路
曝光 → 激发兴趣 → 立即行动 → 到店转化

### 关键动作
1.【优惠钩子】内容里放"到店领礼品/专属价"
2.【地址前置】视频前三秒或封面放门店位置
3.【限时紧迫】设置活动截止日期
4.【POI挂载】抖音视频必须挂载门店地址

### 常见问题排查
| 现象 | 原因 | 对策 |
|------|------|------|
| 播放高但到店低 | 优惠不够/位置远 | 加大到店激励 |
| 评论多但成交少 | 引导购买话术弱 | 评论区置顶购买链接 |
| 到店多但客单低 | 现场导购话术差 | 搭配销售培训 |

## 四、门店每日SOP

【早10点】刷热榜，选话题素材
【中12点】发布当天内容（视频/图文）
【晚20点】回填数据，AI分析，决策是否加热

【每周复盘】
- 找出数据最好的3条 → 找共性复用
- 找出数据最差的3条 → 分析原因规避
